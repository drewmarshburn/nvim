## Quick Commands Reference

### Plugin Management
```vim
:Lazy              " Open plugin manager
:Lazy update       " Update plugins
:Lazy clean        " Remove unused plugins
:LazyExtras        " Browse/install LazyVim extras
```

### LSP
```vim
:LspInfo           " Check LSP status
:Mason             " Manage LSP servers/tools
:ConformInfo       " Check formatter status
```

### GitHub (octo.nvim)
```vim
:Octo pr list      " List PRs
:Octo pr view      " View current PR
:Octo pr comments  " View PR comments
:Octo pr create    " Create draft PR
```

### Database (vim-dadbod)
```vim
:DBUIToggle        " Toggle database UI
:DBUI              " Open database UI
```
